# 🍽️ Restaurant Management System

A modern Restaurant Management Web Application built using **Django**, **MySQL**, **Bootstrap**, **HTML**, **CSS**, and **JavaScript**. The application allows customers to browse meals, manage their cart, place orders, apply coupons, and submit reviews, while administrators can manage menu items, inventory, and customer orders through the Django Admin Panel.

---

- 🔐 User Registration & Login
- 🍔 Browse Meals

## ⚙️ Installation

```bash
git clone https://github.com/NehaM2006/django-restaurant-pos-system.git

cd django-restaurant-pos-system

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
