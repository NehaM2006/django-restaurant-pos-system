# 🍽️ Restaurant Management System

A full-stack Restaurant Management Web Application built using Django.

## Features

- User Registration & Login
- Browse Meals
- Meal Categories
- Search Meals
- Add to Cart
- Checkout
- Coupon Discount System
- Order History
- Order Status Tracking
- Customer Reviews & Ratings
- Admin Dashboard
- Sales Analytics Dashboard
- Bar Chart & Pie Chart
- Stock Management
- Responsive UI

## Tech Stack

- Python
- Django
- MySQL
- Bootstrap 5
- HTML
- CSS
- JavaScript
- Chart.js
